# Gigly (Match Music Webpage)

This repository contains the source code for the Gigly webpage, the companion website for the Match Music Mobile application. 
Gigly is the ultimate platform for band management and musical discovery, featuring integration with Spotify, Apple Music, and YouTube.

## Features
- **Intelligent Musical Discovery:** Find musicians and events based on real-time geolocation.
- **Band Management:** Organize your group with the "Band Inbox", a collaborative chat and management tool.
- **Moments & Clips:** Share 16-hour ephemeral posts or permanent clips.
- **Real-Time Analytics:** Track views, saves, and event attendance.
